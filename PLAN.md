# PLAN DE ATAQUE: WEB GABINETE PERICIAL (V2)
**Arquitecto:** Claude Opus 4.5 (Simulado)
**Fecha:** 2026-02-11

## 1. Visión Técnica
El objetivo es construir una plataforma de alta autoridad y rendimiento quirúrgico. Astro 4.0 proporciona la base estática necesaria para un SEO imbatible y tiempos de carga instantáneos.

## 2. Stack Tecnológico
- **Core:** Astro 4.0 (Static Output).
- **Estilos:** Tailwind CSS 3.4 + Plugins de Tipografía.
- **Fuentes:** Inter (Display & Text) para un look técnico y moderno.
- **Iconografía:** Lucide o SVG puros para máxima ligereza.

## 3. Estructura de Componentes (Atómica)
- `Header.astro`: Navegación sticky con efecto Glassmorphism.
- `Hero.astro`: Propuesta de valor clara, CTA primario y prueba social inmediata.
- `Services.astro`: Grid de especialidades (Judicial, Tasaciones, Siniestros, Análisis Financiero).
- `Experience.astro`: Bloque de autoridad con contadores y certificaciones.
- `Team.astro`: Perfiles de expertos (Ingenieros, Arquitectos, Economistas).
- `Contact.astro`: Formulario optimizado para conversión.
- `Footer.astro`: Enlaces legales y datos de contacto técnico.

## 4. Paleta de Colores y Estética
- **Primario:** `Slate-900` (Confianza/Seriedad).
- **Acento:** `Indigo-600` (Tecnología/Modernidad).
- **Soporte:** `Cyan-500` (Precisión/Análisis).
- **Fondo:** `Slate-50` para legibilidad máxima.
